Revit

A backoffice application for Selor to evaluate candidates based on predefined metrics

View on GitHub Download .zip Download .tar.gz

Project description

For Selor an agency that provides recruiting services for over 150 different governmental institutes, we built an application to facilitate the scoring of candidates in face-2-face interviews.

With Revit, the judges at Selor are able to evaluate any job candidate using any device, whether it's a tablet, a smartphone or a laptop. A huge step up from pencils and paper for both the judges, the candidates and the administrative services at Selor.

Revit is built using open-source components and open development to ensure easy adaptability in the future. The application was designed, built, evaluated and deployed during an intensive 4 week program at open Summer of code, an initiative that allows students to work on real world applications for companies and governmental institutes.

Main benefits

The main benefits are clear from the start. Let's go over the 3 biggest ones.

Fast

No more needless input of existing candidate data.

No more time spent on copying the scores to an electronic system.

Secure

Better overview of inconsistencies and anomalies.

No more missing or blank results.

Adaptable

Last minute changes to a test become a breeze with the built-in edit features.

What is open Summer of code

Open Summer of code -oSoc in short- is a 4 week programme in different Belgian cities during the summer of 2016. The goal: provide Belgian based students the training, network and support necessary to transform open innovation projects into powerful real-world services.

Find out more about oSoc here: 2016.summerofcode.be
